This is a Streamlit web application that predicts whether a movie review is
GOOD or BAD using a Logistic Regression machine learning model.

## Features
- User-friendly interface
- Predicts sentiment of movie reviews
- Displays confidence score
- Built with Streamlit and Scikit-learn

## How it Works
1. Enter a movie review
2. Click "Predict"
3. The model classifies the review as GOOD or BAD

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- TF-IDF Vectorization
- Logistic Regression
